## What is Defensive Security?

Defensive Security focuses on protecting organizations from cyber attacks by identifying threats, monitoring systems, and responding to security incidents.

---

## Blue Team

Blue Team is responsible for defending an organization's infrastructure.

Responsibilities:

- Monitoring security events
- Detecting suspicious activities
- Investigating alerts
- Responding to incidents
- Improving security controls

---

## Security Operations Center (SOC)

A SOC is a team that continuously monitors an organization's security environment.

SOC analysts:

- Analyze security alerts
- Investigate threats
- Detect malicious activities
- Support incident response

---

## Incident Response

Incident Response is the process of handling cybersecurity incidents.

Main stages:

1. Preparation
2. Detection and Analysis
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

---

## Digital Forensics

Digital Forensics is the process of collecting and analyzing digital evidence after a security incident.

It helps in:

- Finding attack methods
- Understanding attacker activities
- Recovering evidence

---

## Security Tools Overview

Common defensive security tools:

- SIEM
- Firewall
- IDS
- IPS
- Antivirus
- Endpoint Detection and Response (EDR)

---

## Key Learnings

- Defensive security protects systems from cyber threats.
- Blue Teams monitor and defend organizations.
- SOC teams investigate security alerts.
- Incident response helps reduce damage from attacks.
- Continuous monitoring is important for cybersecurity.
